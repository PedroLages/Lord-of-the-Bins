# Lord of the Bins - Style Guide

## Design System

### Color Palette
- Primary: Indigo (#4f46e5, #6366f1)
- Background: Slate (#0f172a for dark, #f8fafc for light)
- Accent: Emerald (#10b981), Amber (#f59e0b), Red (#ef4444)
- Text: White/Slate for dark mode, Slate-900 for light mode

### Typography
- Font Family: System default (-apple-system, BlinkMacSystemFont, "Segoe UI", etc.)
- Headings: Bold, larger sizes
- Body: Regular weight
- Code/Numbers: Monospace font

### Spacing
- Use Tailwind's spacing scale (4px base unit)
- Consistent padding/margin throughout

### Component Patterns
- Rounded corners: `rounded-lg` for most elements
- Borders: Subtle, using theme colors
- Shadows: Minimal, only where needed for depth
- Hover states: Slight color shift and/or scale

## Accessibility
- WCAG 2.1 Level AA compliance
- Semantic HTML
- Keyboard navigation support
- Screen reader friendly
- Color contrast ratios meet standards

## Code Style
- React functional components with hooks
- TypeScript for type safety
- Descriptive variable names
- Comments for complex logic only
- Avoid over-engineering

## UX Principles
- Fast, responsive interactions
- Clear visual feedback
- Minimal cognitive load
- Warehouse/logistics-inspired aesthetics
- Playful but professional tone
